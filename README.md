# [Dotan Books] - Full Stack Web Application

## Overview
A comprehensive Full-stack solution featuring a robust **.NET Web API** backend and a dynamic **Angular** frontend. 
This project demonstrates professional software engineering practices, including modular architecture and RESTful API integration.

## Architecture & Repositories
*   **Backend (.NET C#):** Built with **ASP.NET Core**, focusing on scalable business logic and secure data handling.
    https://github.com/Rivka098/Dotan-Books-Server
*   **Frontend (Angular):** A responsive, component-based UI using TypeScript and Angular's latest features.
    https://github.com/Rivka098/Dotan-Books-client

## Key Technical Features
*   **RESTful API Design:** Seamless communication between C# and Angular.
*   **Database Management:** Integration with **SQL Server** for persistent data storage.
*   **Dependency Injection:** Utilizing .NET's built-in DI for cleaner, testable code.
*   **Component Architecture:** Clean and reusable Angular components[cite: 2].
